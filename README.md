# Parcial-I-Computo-1
# Parcial Primer Cómputo - Programación Computacional IV

## Situación problemática

Muchos pequeños negocios no llevan un control digital de sus ventas, lo que provoca errores en cálculos y mala gestión financiera. Esta aplicación permite calcular ventas diarias incluyendo impuestos de forma automática.

Sector enfocado: Pequeños negocios y emprendedores.

---

## Preguntas

### ¿Qué valor agregado tiene el uso de webcomponents a su proyecto?

Permite encapsular la lógica y estructura en un componente reutilizable, facilitando mantenimiento, escalabilidad y organización del código.

### ¿De qué forma manipularon los datos sin recargar la página?

Se utilizó el evento `submit` del formulario junto con `event.preventDefault()` para evitar la recarga, actualizando el DOM dinámicamente.

### ¿De qué forma validaron las entradas de datos?

Se verificó que los campos no estén vacíos, que los valores numéricos sean válidos y mayores que cero antes de realizar cálculos.

### ¿Cómo manejaría la escalabilidad futura en su página?

Separando componentes, usando módulos JS, backend con base de datos y arquitectura basada en componentes reutilizables.
